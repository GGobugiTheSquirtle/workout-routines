## 1. 프로젝트 개요

### 1.1 목적
운동 경험이 있으나 10년 공백 후 맨몸운동을 재시작하는 사용자를 위한 **커리큘럼 기반 운동 앱**. 초급부터 고급까지 단계적으로 진행하며, 계획표 + 체크 + 기록 + 진행도 시각화 + 동기부여를 하나의 앱에서 제공한다.

### 1.2 사용자 프로필
- 174cm / 80kg, 과거 운동 경험자 (10년 공백)
- 현재 체력: 푸시업 15개, 맨몸 스쿼트 50개, 턱걸이 0개 (매달리기 겨우)
- 보유 장비: 치닝디핑, 3단 조절 푸시업바, 스쿼트밴드, 요가매트, 케틀벨 12/16kg

### 1.3 핵심 제약
- **시간**: 하루 30분 이내 (워밍업 5분 + 운동 23분 + 쿨다운 2분)
- **종목 수**: 분할당 4-5종목, 검증된 강력한 종목 위주
- **루틴 패턴**: 2분할 [A-휴-B-휴-휴] 기본, 주말 몰아서 가능하도록 유연성 확보
- **케틀벨 스윙**: 반드시 포함 (입문기 힙힌지 학습 후 초급부터 도입)

### 2.4 데이터 흐름

```
[세트 완료] → LocalStorage 즉시 저장 → 세션 종료 시 Google Sheets 배치 전송
[앱 시작] → Sheets fetch → LocalStorage 캐시 갱신
[오프라인] → LocalStorage만으로 완전 동작 → 온라인 복귀 시 자동 동기화
[충돌 해결] → timestamp 기반 최신 우선 (1인 사용, 충돌 가능성 극히 낮음)
```


## 3. 커리큘럼 & 프로그레션

### 3.1 4단계 로드맵

| 단계 | 기간 | 목적 | 주요 적응 |
|------|------|------|----------|
| **입문** | 0-6주 | 신경근 재활성화, 관절 적응 | 근력 ↑30-40% (신경 기여), 결합조직 강화 |
| **초급** | 7-16주 | 기초근력 확립, 케틀벨 도입 | 제지방량 +1-2kg, 턱걸이 가능 |
| **중급** | 17-32주 | 근비대 집중 | 제지방량 +3-5kg 누적, 근육 기억 효과 정점 |
| **고급** | 33주+ | 고강도 근비대 & 스킬 | 고급 프로그레션, EMOM 스윙 |

**과학적 근거**:
- 입문기 6주 필수: 결합조직(건·인대) 적응에 8-12주 필요 (Magnusson 2010). 근력은 빨리 올라오지만 건은 따라가지 못함
- 근육 기억(myonuclei 보존): 과거 훈련으로 획득한 근핵은 최대 15년 보존 → 재적응 2-3배 빠름 (Gundersen 2016)
- 입문기 RPE 6-7 엄격 준수, 주간 볼륨 증가 10% 이내

### 3.2 루틴 상세

#### 입문 (0-6주) — 신경근 재활성화 & 관절 적응

**A분할: Push + Core**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 인클라인 푸시업 (푸시업바 높은단) | 3×8-12 | 체중 부하 ~40% |
| 딥스 네거티브 (5초 하강) | 3×5 | 어깨 불안정 시 벤치딥 대체 |
| 데드행 (능동적 어깨 팩킹) | 3×15-30초 | 턱걸이 기반 구축 |
| 플랭크 | 3×20-30초 | 코어 안정성 |

**B분할: 하체 + Hinge**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 맨몸 스쿼트 (밴드 착용) | 3×12-15 | 둔근 활성화 |
| 런지 (교대) | 3×8/측 | 편측 안정성 |
| 글루트브릿지 | 3×12-15 | 둔근 약화 보완 |
| 케틀벨 데드리프트 (12kg) | 3×10 | 힙힌지 패턴 학습 (스윙 준비) |

#### 초급 (7-16주) — 기초근력 확립 & 케틀벨 스윙 도입

**A분할: Push + Pull**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 푸시업 (무릎→풀 전환) | 3×8-12 | 더블 프로그레션 |
| 딥스 (어시스트→풀) | 3×5-8 | 어깨 안정 확인 후 |
| 밴드 어시스트 턱걸이 | 3×3-5 | 반동 금지 |
| 행잉 니레이즈 | 3×8 | 코어 + 그립 |

**B분할: 하체 + Swing**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 밴드 스쿼트 (풀 ROM) | 3×12-15 | 깊이 강조 |
| 불가리안 스플릿 스쿼트 | 3×8/측 | 편측 근력 |
| **케틀벨 스윙 (12kg)** | **5×10** | 힙힌지 숙달 후 도입 (S&S 프로토콜) |
| 싱글레그 글루트브릿지 | 3×10/측 | 둔근 편측 강화 |

#### 중급 (17-32주) — 근비대 집중

**A분할: Push + Pull**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 딥 푸시업 (푸시업바) | 4×8-12 | ROM 증가 → stretched position training |
| 딥스 | 4×8-12 | 전경 유지 |
| 턱걸이 (풀그립) | 4×5-8 | 클린 렙 |
| 케틀벨 프레스 (16kg) | 3×8/측 | 어깨 근비대 보충 |

**B분할: 하체 + Posterior**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 피스톨 스쿼트 프로그레션 | 4×5/측 | 단계적 ROM 확장 |
| 밴드 스쿼트 (딥) | 4×12 | 볼륨 확보 |
| **케틀벨 스윙 (16kg)** | **5×15** | 중량 업그레이드 |
| 노르딕컬 네거티브 | 3×5 | 네거티브만, 점진적 |

**시간 최적화**: Antagonist Superset (푸시업↔턱걸이) — 동일 볼륨, 시간 40% 절감 (Robbins 2010)
**마지막 세트 Myo-reps**: Activation set 후 짧은 휴식(5-10초) 미니셋 → 시간 효율 근비대

#### 고급 (33주+) — 고강도 근비대 & 스킬

**A분할: Push + Pull**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 아처 푸시업 | 4×6-8/측 | 편측화 과부하 |
| 웨이티드 딥스 (케틀벨) | 4×6-8 | 외부 부하 추가 |
| L-sit 턱걸이 | 4×6-8 | 코어 통합 |
| 케틀벨 TGU (16kg) | 3×3/측 | 전신 안정성 |

**B분할: 하체 + Posterior**
| 종목 | 세트×횟수 | 비고 |
|------|----------|------|
| 피스톨 스쿼트 (풀 ROM) | 4×5/측 | 완성형 |
| 케틀벨 고블릿 스쿼트 (16kg) | 4×10 | 부하 추가 |
| **케틀벨 스윙 (16kg)** | **6×20 EMOM** | 최고 볼륨 |
| 노르딕컬 (풀 ROM) | 4×5 | 완성형 |

### 3.3 레벨 승급 기준

2주 연속 모든 기준 충족 시 자동 승급 제안 (작업세트 기반, 1RM 테스트 아님)

| 전환 | 푸시업 | 딥스 | 턱걸이 | 스쿼트 | 기타 |
|------|--------|------|--------|--------|------|
| 입문→초급 | 풀 10개 | 네거티브 5초×5 | 데드행 30초 | 20개 풀ROM | — |
| 초급→중급 | 20개 | 풀 8개 | 클린 1개 | 밴드 15개 | 스윙 12kg×10×5 폼OK |
| 중급→고급 | 딥 15개 | 12개 | 8개 | 피스톨 1/측 | 스윙 16kg×15×5 |

### 3.4 프로그레션 모델

**세션 내 — 더블 프로그레션**:
횟수 상한 도달 → 다음 세션 난이도↑ (변형/중량) → 횟수 하한 리셋
예: 푸시업 12회 달성 → 다이아몬드 8회로 전환

**4주 블록 주기화 (DUP 변형)**:
- W1: 적응 (RPE 6-7)
- W2-3: 과부하 (RPE 8-9, 볼륨 +1세트 또는 +2회)
- W4: 디로드 (볼륨 50%, RPE 5-6)

**맨몸운동 과부하 우선순위** (Suprak 2011):
1. 레버리지 조작 (인클라인→플랫→디클라인: 체중 40%→64%→74%)
2. 편측화 (양팔→한팔: 부하 ~80-90% 증가)
3. 템포 조작 (Eccentric 3-5초)
4. ROM 증가 (deficit, 딥 등)
5. 볼륨 증가 (세트/레프 추가)

### 3.5 주간 스케줄

| 월 | 화 | 수 | 목 | 금 | 토 | 일 |
|---|---|---|---|---|---|---|
| **A분할** | 휴식 | **B분할** | 휴식 | 휴식 | 모빌리티(선택) | 완전휴식 |

주말 몰아서 할 경우: 토(A) + 일(B)도 가능하도록 앱에서 유연하게 스케줄 조정 지원.

## 4. UX/UI 설계

### 4.1 화면 구성 (5개 탭)

| 탭 | 주요 기능 |
|----|----------|
| 🏠 Home | 대시보드: 오늘 루틴 카드, 연속기록, 레벨 진행바, 주간 히트맵, 격려 메시지 |
| 💪 Workout | 운동 실행: 세트/횟수 체크, 휴식 타이머, 폼 가이드, RPE 입력, 세션 요약 |
| 📋 History | 기록 조회: 날짜별 이력, 종목별 상세, 캘린더 뷰, 메모 |
| 📈 Progress | 진행도: 종목별 성장 그래프, 단계 진행바, 뱃지 컬렉션, 총 통계 |
| ⚙️ Settings | 설정: 다크/라이트 토글, 스케줄 조정, Sheets 연결, 동기화, 알림 |

### 4.2 운동 실행 화면 (Workout)

- 상단: 현재 단계·분할·진행 상태 (종목 2/4 · 세트 2/3)
- 중앙: 현재 종목명 + 목표 횟수/세트
- 세트 인디케이터: 완료(초록)/현재(파랑)/대기(회색) — 64px 원형
- 횟수 입력: 큰 숫자 표시 + ±버튼 (48px 터치 타겟)
- 하단: "세트 완료" 버튼 (64px+, thumb zone)
- 접이식 폼 가이드 카드 (핵심 큐 3-4개)

### 4.3 휴식 타이머

- 세트 완료 시 자동 시작
- 큰 카운트다운 숫자 (40px+)
- 권장 휴식: 60초(입문) / 90초(근비대)
- +30초 / 건너뛰기 버튼
- 다음 종목 미리보기

### 4.4 대시보드 (Home)

- 인사 + 연속기록 일수
- 주간 히트맵 (GitHub 잔디 스타일): 운동일 초록, 휴식일 회색, 미래 점선
- 오늘 루틴 카드: 분할명, 종목 요약, 예상 시간, "운동 시작하기" CTA
- 레벨 진행바: 현재 Lv, 다음 단계까지 %
- 최근 획득 뱃지

### 4.5 모바일 UX 원칙

- **터치 타겟**: 최소 48×48px, 핵심 액션 64px+
- **Thumb Zone**: 주요 버튼 화면 하단 배치
- **Wake Lock**: `navigator.wakeLock.request('screen')` — 운동 중 자동
- **햅틱**: `navigator.vibrate(50)` — 세트 완료, 타이머 종료
- **큰 폰트**: 운동 중 세트/횟수 24px+, 타이머 40px+
- **다크/라이트**: Pico CSS `data-theme` 토글, localStorage 저장

## 5. 게이미피케이션

### 5.1 XP 시스템

| 행동 | XP |
|------|-----|
| 앱 출석 (운동 시작) | 50 |
| 세트 완료 | 10 |
| 풀 루틴 완료 | 100 |
| 레벨업 보너스 | 500 |
| 복귀 보너스 (3일+ 공백 후) | 2배 |

### 5.2 레벨 시스템

Lv.1-40, 4단계와 연동:
- 입문: Lv.1-10
- 초급: Lv.11-20
- 중급: Lv.21-30
- 고급: Lv.31-40

### 5.3 뱃지

| 뱃지 | 조건 |
|------|------|
| 복귀전사 | 첫 운동 완료 |
| 3일 연속 | streak 3일 |
| 7일 연속 | streak 7일 |
| 30일 연속 | streak 30일 |
| 100일 연속 | streak 100일 |
| 첫 풀업 | 턱걸이 1회 달성 |
| 스윙마스터 | 케틀벨 스윙 도입 |
| 레벨업 | 단계 승급 (입문→초급 등) |

### 5.4 연속기록 (Streak)

- 연속 운동일수 카운터
- **보호막 월 2회**: 빠뜨려도 streak 유지 (자동 사용)
- 끊어져도 "최장 기록 7일 | 현재 0일" 병렬 표시 → 과거 성과 보존
- 마일스톤(7/30/100일) 도달 시 특별 뱃지 + 회고 메시지

### 5.5 격려/피드백 메시지

| 상황 | 메시지 (동반자형 톤) |
|------|---------------------|
| 운동 시작 | "오늘도 왔네요. 이미 절반은 한 겁니다." |
| 세트 완료 | "푸시업 10회 — 지난주보다 2회 늘었습니다." (수치 기반) |
| 루틴 종료 | "총 12세트 완료. 이번 주 3회째입니다." |
| 2일 빠짐 | "쉬는 것도 훈련입니다. 5분 스트레칭만 어때요?" |
| 3일+ 빠짐 | "마지막에 스쿼트 20회를 해냈죠. 거기서 이어갈까요?" |
| 복귀 | 복귀 보너스 XP 2배 + "돌아왔군요!" |

**톤 원칙**: 코치형(지시) ✗ → 동반자형(인정+사실) ✓. 과한 칭찬은 역효과.

### 5.6 진행도 시각화

- **주간 히트맵**: 출석 시각화, 빈칸 채우고 싶은 심리 (Zeigarnik 효과)
- **종목별 성장 그래프**: 꺾은선 차트, 우상향 곡선이 가장 강력한 내적 동기
- **단계 진행바**: 다음 단계까지 % → 목표 근접 효과 (Goal Gradient)
- **절대 비교 금지**: 과거의 나 vs 현재의 나만 표시

## 6. 안전 시스템

### 6.1 6가지 안전 메커니즘

| # | 시스템 | 트리거 | 반응 |
|---|--------|--------|------|
| 1 | 통증 체크 팝업 | 세트 완료 시 (입문기 전체, 이후 신규 종목 첫 2주) | "관절 통증?" → 예 시 중단 + 대체 동작 |
| 2 | RPE 과부하 경고 | RPE 9-10 2회 연속 | "볼륨 감소/디로드 권장" |
| 3 | 볼륨 급증 감지 | 전주 대비 총 세트 20%+ 증가 | "급격한 볼륨 증가 — 부상 위험" |
| 4 | 휴식일 미준수 | 동일 부위 48시간 미경과 | "회복 부족 — 휴식 또는 다른 부위 권장" |
| 5 | 공백 복귀 | 3일+ 미접속 후 복귀 | "마지막 세션의 80%로 시작" + 복귀 XP |
| 6 | 폼 리마인더 | 신규 종목 첫 4주 | 세트 전 폼 큐 카드 자동 표시 |

### 6.2 워밍업 프로토콜 (5분)

| 시간 | 동작 |
|------|------|
| 1분 | 관절 원돌리기 (손목→발목, 각 10회) |
| 1분 | 견갑골 활성화 (월 슬라이드 + 밴드 풀어파트) |
| 1분 | 힙힌지 드릴 (맨몸 굿모닝 10회) |
| 1분 | 동적 스트레칭 (월드 그레이티스트 좌우 3회) |
| 1분 | 첫 종목 경량 세트 (50% 강도 8회) |

### 6.3 쿨다운 프로토콜 (2분)

| 시간 | 동작 |
|------|------|
| 30초 | 호흡 정리 (심호흡 5회) |
| 45초 | 대흉근 스트레칭 (A분할) / 고관절굴곡근 스트레칭 (B분할) |
| 45초 | 추가 타겟 스트레칭 |

### 6.4 단계별 위험 종목 & 대안

| 단계 | 위험 동작 | 위험 요인 | 안전 대안 |
|------|----------|----------|----------|
| 입문 | 딥스 (풀) | 어깨 불안정성 | 벤치딥 또는 네거티브만 |
| 입문 | 풀 푸시업 과다 | 건 과부하 | 인클라인 푸시업바 |
| 초급 | 네거티브 턱걸이 5세트+ | 팔꿈치 건염 | 주 2회, 3-4회/세트 제한 |
| 중급 | 노르딕컬 | 햄스트링 파열 | 네거티브만, 점진적 ROM |
| 전 단계 | 케틀벨 스윙 30+회 | 요추 피로 폼 붕괴 | 10-15회 짧은 세트 |

### 6.5 10년 공백자 특별 주의

- 근력은 4-6주 회복되지만 **건·인대 적응은 12-16주**
- 근력이 먼저 올라와 "할 수 있다"고 느끼는 시점이 가장 위험
- 입문기 8주: RPE 6-7 엄격 준수
- 주간 볼륨 증가 10% 이내
- 팔꿈치 건염, 어깨 충돌 증후군, 슬개건염 특별 주의

## 7. 종목별 폼 가이드

### 푸시업
1. 손 위치: 어깨 바로 아래, 손가락 전방
2. 팔꿈치 45도 이내 (화살표 형태, T자 금지)
3. 코어 브레이싱 — 골반 처짐/들림 없이 플랭크 유지
4. 흉곽 바닥 터치까지 Full ROM

### 딥스
1. 어깨 내림(depression) 유지, 으쓱 금지
2. 하강 시 팔꿈치 90도까지만 (과도한 ROM → 어깨 충돌)
3. 약간 전경(forward lean)으로 흉근 활성화
4. 상단에서 soft lockout (완전 잠금 금지)

### 턱걸이 (및 준비동작)
1. 데드행 시 능동적 어깨 팩킹 (scapular depression + retraction)
2. 그립: 어깨너비, 엄지 감싸쥐기
3. 반동(kipping) 절대 금지 — 회전근개 부상 주원인
4. 네거티브: 3-5초 제어된 하강

### 스쿼트
1. 발 어깨너비, 발끝 15-30도 외회전
2. 무릎은 발끝 방향 추적 (valgus 금지)
3. 힙힌지 먼저 시작, 허리 중립 유지
4. 밴드 사용 시 외회전 텐션 의식

### 케틀벨 스윙
1. 힙힌지가 핵심 — 스쿼트 패턴 아님
2. 팔이 아닌 둔근/햄스트링 폭발력으로 추진
3. 코어 브레이싱 + 글루트 스퀴즈로 상단 잠금
4. 12kg부터 시작, 16kg는 최소 4주 후
